# **App Name**: ThoughtFlow

## Core Features:

- Calendar-Based Daily Journaling: Navigate and create daily journal entries through an intuitive calendar interface, with visual indicators for entry status.
- Rich Block-Based Editor: Utilize a flexible editor for writing entries, supporting various content blocks like headings, lists, quotes, code blocks, and toggle blocks.
- AI-Powered Topic Detection: An AI tool that automatically identifies and extracts key topics and recurring themes from journal entries, facilitating pattern recognition.
- Dynamic Idea Mindmap: Visually represent connected ideas and topics from entries in an interactive mindmap, with nodes indicating topics and links showing relationships. Node size and line thickness reflect frequency and connection strength.
- Entry Search & Filtering: Search past journal entries by keywords and filter them based on detected topics or dates to quickly retrieve information.
- User Authentication: Secure user login and registration process to protect individual journal data and ensure a personalized experience.
- Appearance Mode Toggle: Switch between a dark and a light visual theme for the application to suit user preference and environmental lighting.

## Style Guidelines:

- The visual concept embraces 'deep thinking' and 'tech innovation', influencing a choice of a dark color scheme. The primary color, representing clarity and digital intelligence, is Electric Blue (#3B82F6).
- The background color is a subtle Dark Charcoal with a hint of blue (#191B1E), derived from the primary hue, providing depth and minimal distraction.
- An energetic accent color, Cyan Green (#06D6EE), is used for highlights and interactive elements, providing a strong contrast to the primary and background colors and supporting the tech-forward aesthetic.
- Body and headline font: 'Plus Jakarta Sans' (sans-serif) for its modern, clear, and highly legible appearance, suitable for dense textual content and bold headlines alike. Note: currently only Google Fonts are supported.
- Utilize functional and clean 'Material Symbols Outlined' for system-level actions and visual cues, aligning with the app's minimalist yet informative interface.
- A clean, mobile-first layout prioritizing content readability and intuitive navigation. Elements often utilize a 'glassmorphism' effect on darker backgrounds for depth without visual clutter.
- Subtle and fluid animations are employed for UI transitions and interactive elements, particularly for dynamic elements like mindmap node interactions and loading states, ensuring a responsive user experience.
